## Primeiros passos

Siga este fluxo para configurar seu primeiro projeto e conectar um canal.

<Steps>
<Step title="Autenticar com Google">
Faça login via Google OAuth 2.0. Não há cadastro de senha na plataforma.
</Step>
<Step title="Criar um projeto">
No painel principal, clique em criar projeto e informe um nome.
</Step>
<Step title="Configurar o canal">
Preencha o prompt de nicho, estilo de vídeo, tempo e número de cortes.
</Step>
<Step title="Conectar uma plataforma">
Use o Flow Builder para adicionar um node e autenticar a conta via OAuth.
</Step>
<Step title="Executar o pipeline">
Inicie a produção e acompanhe o progresso pelos logs.
</Step>
</Steps>

## Criar um projeto

<Note>
  O nome do projeto é a unidade central do fluxo de automação.
</Note>

<CodeGroup>
<CodeGroup.Code title="JavaScript (fetch)">
```js
fetch('/panel/create/project', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ name: 'Canal de Tecnologia' })
});
```
</CodeGroup.Code>
<CodeGroup.Code title="Python">
```py
import requests

response = requests.post(
  'https://seusite.com/panel/create/project',
  json={'name': 'Canal de Tecnologia'}
)
```
</CodeGroup.Code>
</CodeGroup>

## Conectar canal

Use a rota de conexão para iniciar o fluxo OAuth.

```http
POST /panel/channel/connect?platform=youtube
```

<Info>
  O token OAuth é armazenado criptografado com AES-256. A plataforma não guarda senhas.
</Info>

## Recomendações iniciais

- Confirme seu saldo de créditos antes de iniciar a produção.
- Configure o `channel_prompt` com o máximo de detalhe possível.
- Use `research_topics` se quiser conteúdo alinhado a tendências.
