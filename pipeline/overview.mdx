## Pipeline de IA

O pipeline do PixelJamCreator Studio é composto por cinco etapas sequenciais:

1. Geração do roteiro
2. Criação de imagens
3. Geração de música
4. Edição e renderização
5. Publicação

## Fluxo de produção

- O `channel_prompt` define o propósito do vídeo.
- O sistema divide o vídeo em cortes com base em `amount_cut`.
- Cada etapa executa uma parte do workflow até a publicação.

<Note>
  A execução do pipeline é controlada pelo projeto e consome créditos conforme o uso.
</Note>
